Roomzin Tutorials

Quickstart examples for integrating Roomzin with Java, Go, Python, and .NET SDKs.

Java SDK

Install via Maven:

<dependency>
  <groupId>com.roomzin</groupId>
  <artifactId>rz-java</artifactId>
  <version>0.1.0</version>
</dependency>

Sample usage:

import com.roomzin.Client;

public class Example {
    public static void main(String[] args) {
        Client client = new Client("http://localhost:8080");
        var result = client.queryAvailability("NYC", "2025-06-01", "2025-06-07");
        System.out.println(result);
    }
}

Go SDK

Install:

go get github.com/roomzin/rz-go

Sample usage:

package main

import (
    "fmt"
    "github.com/roomzin/rz-go"
)

func main() {
    client := rz.NewClient("http://localhost:8080")
    result, _ := client.QueryAvailability("NYC", "2025-06-01", "2025-06-07")
    fmt.Println(result)
}

Python SDK

Install:

pip install rz-py

Sample usage:

from roomzin import Client

client = Client("http://localhost:8080")
result = client.query_availability("NYC", "2025-06-01", "2025-06-07")
print(result)

.NET SDK

Install:

dotnet add package RzDotNet --version 0.1.0

Sample usage:

using Roomzin;

class Program {
    static void Main(string[] args) {
        var client = new Client("http://localhost:8080");
        var result = client.QueryAvailability("NYC", "2025-06-01", "2025-06-07");
        Console.WriteLine(result);
    }
}
