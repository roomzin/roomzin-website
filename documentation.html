Getting Started

Roomzin is designed for seamless integration with your existing booking platform infrastructure. Our smart SDKs handle the complexity so you can focus on your business logic.

Available SDKs

Enterprise-ready Java client with automatic cluster discovery

implementation 'com.roomzin:client:1.0.0'

High-performance Go client for microservices architecture

go get github.com/roomzin/go-client

Python SDK

Python client for data science and integration scenarios

pip install roomzin-client

.NET client for enterprise Windows environments

Install-Package Roomzin.Client

Quick Integration Guide

1. Initialize Client

// Java example
RoomzinClient client = RoomzinClient.builder()
    .endpoint("roomzin-cluster:8080")
    .build();

2. Define Your Models

// Property inventory model
public class PropertyInventory {
    private String propertyId;
    private List roomTypes;
    private Map availability;
    // ... getters and setters
}

3. Perform Queries

// Single query for complex searches
SearchRequest request = SearchRequest.builder()
    .location(geoPoint)
    .checkIn(checkInDate)
    .checkOut(checkOutDate)
    .roomCount(2)
    .amenities(Arrays.asList("wifi", "pool"))
    .build();

SearchResponse response = client.search(request);

Key Integration Features

Single-Query Model

Complex searches (geo, amenities, categories, stars, cancellation rules, date ranges, pricing filters) handled in one request without in-memory joins.

Automatic Cluster Awareness

SDKs automatically discover cluster topology, handle failover, and route requests efficiently without manual configuration.

Zero-Downtime Operations

Clusters continue serving traffic while saving snapshots and performing maintenance operations.

Effortless Bootstrapping

Load millions of records in seconds with 0.4M/s initial load speed and efficient memory utilization.

REST API Reference

For management and monitoring, Roomzin provides a comprehensive REST API:

GET /health

Cluster health status and node information

GET /metrics

Performance metrics and cluster statistics

POST /admin/snapshot

Trigger manual snapshot creation

GET /admin/nodes

Cluster node information and status
